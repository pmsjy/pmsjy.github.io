    产品定义 #

    • BIM技能实训系统是服务BIM类训练、考试、竞赛的平台。
    • 满足各类BIM软件教学、1+X考前训练、BIM类技能竞赛的训练和模拟需求，解决教师BIM模型评分问题，帮助学生自查自纠。

    产品优势 #

    • 价值一：满足传统课程升级需求，减轻老师教学评价负担
      • 产品可实现Revit土建、机电、BIM土建算量、BIM安装算量、施工策划、模板、脚手架、5D软件的自动评分，传统课程升级过程中所涉及的上述软件，都可满足实训、考试要求。
    • 价值二：丰富的云题库资源，多方位满足教学需求
      • 云题库包含客观题一千余道（单选题和多选题），实操题百余道，涉及所有BIM类教学软件，1+X初级建模、中级结构工程、工程管理、建筑设备历届真题、BIM类竞赛真题。
    • 价值三：教师可自定义设置评分比例，学生可自主查漏补缺
      • 教师可对每一道题目自定义设置评分比例，根据自身需求，跨软件、跨专业自定义组卷。考后可对比三维模型，辅助人工阅卷。学生参与训练和考核后可查看评分报告，自主校对标准答案，自主查漏补缺。
    • 价值四：防作弊诚信考试，满足万人同时在线考试
      • 学生在参与训练考试的过程中，通过摄像头、屏幕监管、霸屏控制、操作记录等手段实现防作弊考试闭环。

    竞品相比较的优势 #

    • Revit软件自动评分：相关友商，没有Revit自动评分，南京的慧筑的Revit标准模型无法复用，老师出题操作繁琐，学生考后无法自主校对标准答案
    • 对比部分友商只有评分功能，没有考试的流程性
    • 云题库资源丰富：品茗云题库涉及面广，真题内容多。对比其他友商产品缺少真题

    如何入课 #

    • 《Revit基础建模》、《施工组织设计》、《施工技术应用》、《工程概预算》相应的课程期末实训或期末考试都会用到BIM软件，都可辅助老师完成教学评价。
    • 1+X（BIM）、BIM类竞赛训练，学生自主刷理论题、实操题，自主评分，查漏补缺。
